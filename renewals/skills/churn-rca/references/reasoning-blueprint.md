# Reasoning Blueprint

This skill applies structured reasoning to root cause analysis of confirmed customer churn events and cohort-level churn pattern analysis.

## D1 — Cognitive Stance
CLASSIFY the request type and select the appropriate RCA mode (individual analyze, cohort pattern analysis, or export).
CONSTRAINTS: Distinguish correlation from causation; flag data quality issues explicitly; never assign root cause without named evidence.
EXPERT: Approach this as a customer success analyst with churn modeling expertise and pattern recognition across portfolio-level data.
