# rev-ops.outcome-statement-builder

Builds structured, verifiable outcome statements using the Seven-Stage Value Chain. Two-phase with human approval gate between phases. Produces a Markdown Outcome & Value Registry and an HTML review deck for cross-functional ratification. Inputs are product capability descriptions or draft outcome statements.

## Use it for

- Transform product/service capabilities into structured outcome statements
- Apply Seven-Stage Value Chain to each capability
- Produce Outcome & Value Registry (Markdown)
- Generate HTML review deck for Sales/Marketing/RevOps/CS ratification
- Gate approval between Phase 1 (structure) and Phase 2 (registry + deck)

## Don't use it for

- Tracking outcome achievement post-close (use deal-to-outcome-tracing)
- Value tracking per account (use outcome-to-value-tracking)
- CRM edits

## How to trigger it

Say something like:

- "outcome statement"
- "build outcome statements"
- "Seven-Stage Value Chain"
- "outcome registry"
- "product capabilities to outcomes"

## What you get

- Outcome & Value Registry (Markdown)
- HTML review deck for cross-functional ratification
- Phase 1 outcome structure document (approval gate output)

## Prerequisites

- Product capability descriptions or draft outcome statements
- Target audience personas for value chain mapping

## Governance

**Approval required** — output must be reviewed before distribution.
- Two-phase with explicit human approval gate between phases
- Phase 2 does not proceed without Phase 1 approval
