# csm.success-plan-canvas

Builds a structured success plan canvas — a one-page view of account goals, success metrics, key milestones, and mutual commitments. Supports initial canvas creation at onboarding and refresh at major lifecycle transitions.

## Use it for

- Initial success plan canvas for new accounts
- Success plan refresh at renewal or expansion milestone
- Mutual success commitment documentation

## Don't use it for

- Progress review against an existing plan (use csm.success-plan-progress-review)
- Full QBR construction (use csm.qbr-builder)

## How to trigger it

Say something like:

- "success plan"
- "build a success plan"
- "mutual success plan"
- "success criteria canvas"
- "onboarding success plan"

## What you get

- Success plan canvas (structured markdown)

## Prerequisites

- csm CLAUDE.md
- Account goals and success criteria

## Governance

- Canvas requires CSM review before sharing with customer
