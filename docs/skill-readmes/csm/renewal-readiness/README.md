# csm.renewal-readiness

Assesses renewal readiness 90–180 days before renewal date. Evaluates health, relationship strength, commercial risk, and success plan completion. Produces readiness brief, renewal timeline, and customer-facing renewal summary.

## Use it for

- Renewal readiness assessment (90–180 day window)
- Renewal timeline construction
- Customer-facing renewal summary

## Don't use it for

- Negotiation strategy (use renewals.negotiation-prep)
- Renewal forecasting (use renewals.renewal-forecast)
- Risk assessment post-signal detection (use renewals.risk-assessment)

## How to trigger it

Say something like:

- "renewal readiness"
- "prepare for renewal"
- "renewal assessment"
- "is [account] ready to renew"

## What you get

- Renewal readiness brief
- Renewal timeline
- Customer-facing summary

## Prerequisites

- csm CLAUDE.md
- Renewal date and account context

## Governance

- Customer-facing summary requires CSM review before sending
