# csm.value-statement

Generates value realized narratives in multiple formats — internal CS summary, customer-facing statement, executive brief, and AE handoff. Draws on product usage, business outcomes, and outcome catalog alignment.

## Use it for

- Internal value summary for CS leadership
- Customer-facing value statement
- Executive brief for sponsor relationship
- AE handoff value narrative

## Don't use it for

- Full QBR document (use csm.qbr-builder)
- Renewal negotiation framing (use renewals.negotiation-prep)

## How to trigger it

Say something like:

- "value statement"
- "value realized"
- "what value have we delivered"
- "value narrative"
- "ROI summary"

## What you get

- Value statement in requested format

## Prerequisites

- csm CLAUDE.md
- Usage and outcome data

## Governance

- Customer-facing outputs require CSM review
- ARR figures flagged for Finance review in executive variants
