# cs-ops.metric-dashboard

Generates CS metrics dashboards across reporting cadences — weekly, monthly, quarterly, board-level, and CSM performance views. Aggregates health, renewal, onboarding, and engagement metrics into structured reports.

## Use it for

- Weekly team metrics snapshot
- Monthly leadership report
- Quarterly business review data package
- Board-level CS metrics summary
- CSM individual performance view

## Don't use it for

- Revenue-side GTM metrics (use rev-ops.gtm-unified-metrics-pulse)
- Real-time dashboard tooling (produces static reports)

## How to trigger it

Say something like:

- "metrics dashboard"
- "weekly metrics"
- "monthly report"
- "board metrics"
- "CSM performance"

## What you get

- Metrics report (structured markdown)
