# csm.success-plan-progress-review

Reviews progress against an existing success plan — generates a milestone scorecard, OCV outcome ratings, and a QBR pre-work note. Designed as the downstream consumer of csm.success-plan-canvas in the success plan chain.

## Use it for

- Milestone scorecard against success plan commitments
- OCV outcome rating update
- QBR pre-work note generation
- Progress narrative for customer-facing review

## Don't use it for

- Building a new success plan (use csm.success-plan-canvas)
- Full QBR construction (use csm.qbr-builder)

## How to trigger it

Say something like:

- "success plan review"
- "progress against success plan"
- "scorecard"
- "how are we tracking"
- "QBR pre-work"

## What you get

- Progress review with milestone scorecard
- QBR pre-work note

## Prerequisites

- csm CLAUDE.md
- Existing success plan canvas
- Current milestone and outcome data

## Governance

- Requires an existing success plan canvas as input
