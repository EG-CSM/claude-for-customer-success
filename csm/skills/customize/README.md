# csm.customize

Updates specific sections of the CSM practice profile without a full re-interview. Alias for cold-start-interview when profile is entirely missing. Shows current values, confirms changes before writing.

## Use it for

- Update a single profile section
- Reset a section to defaults
- Show current CSM configuration

## Don't use it for

- First-time setup (use csm.cold-start-interview)

## How to trigger it

Say something like:

- "update csm config"
- "change my motion type"
- "show csm settings"

## What you get

- Updated csm/CLAUDE.md

## Prerequisites

- Existing csm/CLAUDE.md

- csm.cold-start-interview
