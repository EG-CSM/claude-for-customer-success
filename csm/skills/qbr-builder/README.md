# csm.qbr-builder

Builds and reviews Quarterly Business Reviews. Drafts value delivered, metric performance, next-period priorities, and success plan alignment. Produces internal draft, executive brief, and full QBR document variants.

## Use it for

- Draft a QBR document from scratch
- Review and improve an existing QBR draft
- Generate an executive brief version

## Don't use it for

- Call prep for QBR meeting (use csm.call-prep)
- Value statement narrative only (use csm.value-statement)

## How to trigger it

Say something like:

- "build QBR"
- "draft QBR"
- "quarterly business review"
- "QBR document"
- "review my QBR"

## What you get

- QBR document (markdown)
- Executive brief variant

## Prerequisites

- csm CLAUDE.md
- Account metrics and value delivered data
