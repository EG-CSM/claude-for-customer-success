# csm.call-prep

Generates a pre-call preparation brief calibrated to call type — kickoff, QBR, health review, renewal, check-in, or custom. Includes agenda, talking points, account context, and open action items. Intended for use 24–48 hours before calls.

## Use it for

- Kickoff call preparation
- QBR preparation brief
- Health review call prep
- Renewal conversation prep
- Check-in call agenda

## Don't use it for

- Full QBR document construction (use csm.qbr-builder)
- Renewal negotiation brief (use renewals.negotiation-prep)

## How to trigger it

Say something like:

- "prep for my call"
- "call prep"
- "prepare for QBR"
- "kickoff prep"
- "renewal call prep"

## What you get

- Call prep brief with agenda and talking points

## Prerequisites

- csm CLAUDE.md
- Account name and call type
